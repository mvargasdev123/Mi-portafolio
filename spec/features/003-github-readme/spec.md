# Spec: GitHub Profile README

## Descripción
Creación de un README.md a nivel de perfil de GitHub (repositorio con el mismo nombre de usuario) que actúe como página de aterrizaje profesional para reclutadores y líderes técnicos.

## Requerimientos Funcionales y Visuales
- **Cabecera Dinámica:** Un saludo profesional destacando el rol de "Full-Stack Developer" y el enlace principal al portafolio `mvargasdev.online`.
- **Stack Tecnológico:** Uso de insignias (badges) visuales para listar:
  - Frontend: React, JavaScript, HTML5, CSS3.
  - Backend: Python, FastAPI.
  - Infraestructura/BD: AWS, Docker, Arch Linux, SQLite/PostgreSQL.
- **Proyectos Destacados:** Sección con enlaces a los repositorios clave (API del Portafolio, Web del Portafolio, Asocolgi, OrusStyle).
- **Estadísticas:** Integración de tarjetas dinámicas de GitHub (GitHub Stats Cards) para mostrar commits y lenguajes más usados.

## Criterios de Aceptación
- El archivo debe estar escrito íntegramente en Markdown limpio.
- Todos los enlaces deben funcionar y apuntar a los repositorios correctos.
- Visualmente atractivo, sin bloques de texto aburridos y excesivamente largos.
