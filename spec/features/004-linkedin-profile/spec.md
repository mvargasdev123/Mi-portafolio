# Spec: Optimización del Perfil de LinkedIn

## Descripción
Estructuración y redacción profesional del perfil de LinkedIn de Miguel Vargas para maximizar la visibilidad ante reclutadores IT y líderes técnicos. El perfil debe reflejar coherencia con el portafolio `mvargasdev.online` y el README de GitHub.

## Requerimientos Funcionales y de Contenido
- **Titular (Headline):** Debe incluir el rol "Full-Stack Developer", tecnologías clave (Python, FastAPI, React, AWS) y una propuesta de valor, huyendo de títulos genéricos como "Estudiante".
- **Acerca de (About):** Resumen profesional en formato "Elevator Pitch". Debe mencionar la capacidad de manejar tanto la lógica de negocio (Backend/Bases de datos) como la interfaz de usuario (Frontend), además de habilidades en infraestructura (Linux, Docker).
- **Experiencia / Proyectos:** Las entradas deben estar redactadas orientadas a logros y tecnologías (ej. "Desarrollé la API de Asocolgi usando FastAPI..."). Se incluirán OrusStyle y el propio portafolio.
- **URL Personalizada:** Asegurar que la URL de LinkedIn sea limpia (ej. `linkedin.com/in/mvargasdev`).

## Criterios de Aceptación
- Tono profesional, seguro y orientado a resultados.
- Inclusión estratégica de palabras clave (keywords) para aparecer en los resultados de búsqueda de los reclutadores.
- Textos listos para ser copiados y pegados directamente en la plataforma.
