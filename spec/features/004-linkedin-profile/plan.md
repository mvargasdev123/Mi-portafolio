# Plan de Implementación: Perfil de LinkedIn

## Enfoque Técnico
A diferencia de otras features, esta no implica generar código fuente, sino redacción estratégica ("Copywriting" técnico). El agente actuará como un consultor de marca personal, generando los bloques de texto optimizados basados en la información de la constitución del proyecto.

## Archivos a Modificar / Crear
- `spec/features/004-linkedin-profile/linkedin-copy.md` (Archivo donde el agente volcará los textos generados).

## Secuencia de Acción
1. Analizar el `tech-stack.md` y `mission.md` para extraer palabras clave.
2. Redactar 3 opciones de Titular (Headline) para elegir la mejor.
3. Redactar la sección "Acerca de" estructurada en párrafos cortos y viñetas.
4. Redactar las descripciones para la sección de Experiencia/Proyectos (Portafolio, Asocolgi, OrusStyle).
5. Listar las habilidades (Skills) clave que deben ser añadidas a la plataforma.
