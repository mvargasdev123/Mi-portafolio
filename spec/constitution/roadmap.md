# Roadmap: Portafolio mvargasdev.online

Este documento define el orden de implementación de las características (features) del proyecto mediante la metodología Spec-Driven Development (SDD).

1. **001-portafolio-api**: Diseño y creación de los endpoints del backend en FastAPI (Base de datos, modelos SQLModel y rutas).
2. **002-portafolio-web**: Diseño y creación de la interfaz de usuario interactiva en React y su conexión con la API.
3. **003-github-readme**: Creación del README de perfil de GitHub (Developer Portfolio) estructurado para destacar el stack tecnológico y los proyectos principales.
4. **004-linkedin-profile**: Optimización del perfil de LinkedIn, extrayendo la información clave del portafolio para atraer reclutadores.
