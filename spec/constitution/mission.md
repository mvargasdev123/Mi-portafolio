# Misión del Proyecto: Portafolio Full-Stack (mvargasdev.online)

## 1. Propósito del Proyecto
El objetivo principal de este proyecto es diseñar, desarrollar y desplegar el portafolio web profesional de Miguel Vargas bajo el dominio personalizado `mvargasdev.online`. Este sitio web actuará como la carta de presentación definitiva ante empresas, reclutadores y líderes técnicos del sector tecnológico.

## 2. Propuesta de Valor & Posicionamiento
El portafolio debe demostrar visual y técnicamente la capacidad de adaptación a arquitecturas de software modernas, abarcando tanto el desarrollo Frontend como el Backend de forma fluida. Se hará especial énfasis en:
- **Calidad de Código:** Implementación estricta de estándares de producción, patrones de diseño y principios SOLID.
- **Experiencia de Usuario:** Interfaz profesional, limpia y responsiva basada en las normativas y fundamentos del diseño UI/UX.
- **Competencia Full-Stack Real:** Demostración de habilidades en infraestructura (Arch Linux/servidores locales), bases de datos en la nube (AWS Free Tier), lógica de negocio avanzada (FastAPI) y vistas interactivas (React).

## 3. Público Objetivo
- **Líderes Técnicos y CTOs:** Interesados en la arquitectura del backend, uso de Docker, tipado en Python y limpieza del repositorio.
- **Reclutadores de IT (Recruiters):** Que buscan un perfil dinámico, organizado, con proyectos desplegados en la nube y documentación impecable.

## 4. Criterios de Éxito Técnico
- El sitio debe cargar de forma eficiente y ser 100% responsivo (móvil y escritorio).
- Todo el código debe estar versionado y conectado con GitHub siguiendo la metodología SDD.
- Los proyectos expuestos (Asocolgi, OrusStyle, notas de Obsidian y la configuración de servidores) deben integrarse de manera cohesiva y profesional.
